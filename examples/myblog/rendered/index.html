The File Name becomes your url

Use the formatting you choose, and you can be explicit by using the this.markup configuration.

Images can be inserted by using urlTo such as including the below image:

@!urlTo("media/images/beware.jpg")!@

Source can be included, and formatted inline:

print "Foo!"

through the use of pygments for syntax highlighting.

Files can be read into your page output by using scripts in your site "namespace" such as with: 

<!--(block | cubescript | syntax("py"))-->
$!readFrom("_somecode.py")!$
<!--(end)-->

Execution:

import cubictemp
print cubictemp.File(
    "template",
    foo=3,
    mydict=dict(key="value")
)
<!--(end)-->
