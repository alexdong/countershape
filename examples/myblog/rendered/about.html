My Example Blog

Blog entries are stored in a separate directory tree from the rest of the website.

--| site-src
  +--| site-directory1
     | site-directory2
  | blog-entries
  +--| blog-directory1
     | blog-directory2

This let's us create a website, in which the blog entries are woven.
